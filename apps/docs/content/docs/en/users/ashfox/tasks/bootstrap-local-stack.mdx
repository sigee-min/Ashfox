# Bootstrap Local Service Stack

## Goal

Bring up the Ashfox service stack locally with predictable startup order.

## Steps

1. Install dependencies at repository root (`npm install`).
2. Build required packages (`npm run build`, `npm --workspace @ashfox/web run build`).
3. Start gateway and worker services.
4. Validate endpoint reachability from an external client.
