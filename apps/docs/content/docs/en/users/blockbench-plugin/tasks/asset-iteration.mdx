# Run Modeling and Texture Iteration

## Goal

Iterate asset geometry and textures quickly with repeatable tool calls.

## Steps

1. Ensure project context is initialized (`project.ensure_project`).
2. Apply modeling changes (`add_*`, `update_*`, `delete_*` model tools).
3. Read and assign textures (`read_texture`, `assign_texture`).
4. Preview and inspect intermediate result (`project.render_preview`).
