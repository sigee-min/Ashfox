# 로컬 플러그인 세션 준비

## 목표

Blockbench에서 Ashfox 플러그인을 로드하고 로컬 엔드포인트로 첫 MCP 호출을 성공시킵니다.

## 단계

1. 저장소 루트에서 `npm run build`로 번들을 생성합니다.
2. Blockbench 플러그인 관리자에서 `dist/ashfox-bbplugin.js`를 로드합니다.
3. 플러그인 설정에서 엔드포인트(`/mcp`)를 지정합니다.
4. `system.list_capabilities` 같은 경량 호출로 연결을 확인합니다.
