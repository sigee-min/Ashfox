# 로컬 서비스 스택 부트스트랩

## 목표

Ashfox 서비스 구성을 로컬에서 재현하고 기동 상태를 확인합니다.

## 단계

1. 저장소 루트에서 의존성을 설치합니다 (`npm install`).
2. 필요한 빌드를 수행합니다 (`npm run build`, `npm --workspace @ashfox/web run build`).
3. gateway와 worker 서비스를 실행합니다.
4. 외부 클라이언트 기준으로 엔드포인트 도달성을 확인합니다.
