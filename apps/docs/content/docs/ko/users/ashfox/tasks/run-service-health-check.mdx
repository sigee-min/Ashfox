# 서비스 헬스체크 실행

## 목표

릴리즈 또는 배포 변경 전에 운영 드리프트를 조기에 감지합니다.

## 단계

1. 저장소 품질 게이트를 실행합니다 (`npm run quality`).
2. 문서 품질 검사를 실행합니다 (`npm run docs:quality`).
3. docs 정적 빌드를 실행합니다 (`npm --workspace apps/docs run build`).
4. 정적 정책 게이트를 확인합니다 (`npm run quality:check`).
