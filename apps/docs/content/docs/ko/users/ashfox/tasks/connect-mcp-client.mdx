# 외부 MCP 클라이언트 연결

## 목표

Blockbench 외부 클라이언트에서도 MCP 연결을 안정적으로 확보합니다.

## 단계

1. gateway 엔드포인트 경로와 인증 조건을 확인합니다.
2. 핸드셰이크와 capability 조회(`system.list_capabilities`)를 실행합니다.
3. 읽기 전용 프로젝트 호출로 페이로드 경로를 확인합니다.
4. 클라이언트 로그에서 오류 매핑/재시도 동작을 점검합니다.
