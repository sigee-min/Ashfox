## 빠른 요약
대상 포맷으로 결과물을 내보내 파이프라인 후속 단계에 전달합니다.

## 개요
- 도구 이름: `export`
- 카테고리: 프로젝트
- Strict schema: `true`

## 도구 설명
대상 포맷으로 결과물을 내보내 파이프라인 후속 단계에 전달합니다.

## 입력 스키마 (JSON)
```json
{
  "type": "object",
  "required": [
    "format",
    "destPath"
  ],
  "additionalProperties": false,
  "properties": {
    "format": {
      "enum": [
        "java_block_item_json",
        "gecko_geo_anim",
        "animated_java",
        "generic_model_json",
        "gltf",
        "native_codec"
      ]
    },
    "codecId": {
      "type": "string"
    },
    "destPath": {
      "type": "string"
    },
    "options": {
      "type": "object",
      "additionalProperties": false,
      "properties": {
        "fallback": {
          "type": "string",
          "enum": [
            "strict",
            "best_effort"
          ]
        },
        "includeDiagnostics": {
          "type": "boolean"
        }
      }
    },
    "includeState": {
      "type": "boolean"
    },
    "ifRevision": {
      "type": "string"
    }
  }
}
```

## 필드 레퍼런스
| Field | Type | 필수 | 설명 | 제약 |
| --- | --- | --- | --- | --- |
| `format` | `enum("java_block_item_json", "gecko_geo_anim", "animated_java", "generic_model_json", "gltf", "native_codec")` | ✅ | 스키마에 별도 설명이 없습니다. | enum: "java_block_item_json", "gecko_geo_anim", "animated_java", "generic_model_json", "gltf", "native_codec" |
| `codecId` | `string` | ❌ | 스키마에 별도 설명이 없습니다. | 추가 제약이 선언되지 않았습니다. |
| `destPath` | `string` | ✅ | 스키마에 별도 설명이 없습니다. | 추가 제약이 선언되지 않았습니다. |
| `options` | `object` | ❌ | 스키마에 별도 설명이 없습니다. | additionalProperties=false |
| `options.fallback` | `enum("strict", "best_effort")` | ❌ | 스키마에 별도 설명이 없습니다. | enum: "strict", "best_effort" |
| `options.includeDiagnostics` | `boolean` | ❌ | 스키마에 별도 설명이 없습니다. | 추가 제약이 선언되지 않았습니다. |
| `includeState` | `boolean` | ❌ | 스키마에 별도 설명이 없습니다. | 추가 제약이 선언되지 않았습니다. |
| `ifRevision` | `string` | ❌ | 스키마에 별도 설명이 없습니다. | 추가 제약이 선언되지 않았습니다. |

## 최소 요청 예시
```json
{
  "format": "java_block_item_json",
  "destPath": "./out/model.json"
}
```

## 응답 예시
이 응답은 스키매틱 예시입니다. 실제 값과 선택 필드는 프로젝트 상태 및 런타임 버전에 따라 달라질 수 있습니다.

```json
{
  "structuredContent": {
    "path": "value"
  }
}
```

## 운영 노트
- 입력 스키마에 선언된 필드만 전송하세요.
- 응답 예시는 스키매틱이므로 선택 필드는 방어적으로 처리하세요.
- 변경 도구에서 지원되는 경우 `ifRevision`을 사용해 충돌을 줄이세요.
