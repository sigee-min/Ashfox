## 빠른 요약
대상 포맷으로 결과물을 내보내 파이프라인 후속 단계에 전달합니다.

## 개요
- 도구 이름: `export`
- 카테고리: 프로젝트
- Strict schema: `true`

## 도구 설명
대상 포맷으로 결과물을 내보내 파이프라인 후속 단계에 전달합니다.

## 입력 스키마 (JSON)
```json
{
  "type": "object",
  "required": [
    "format",
    "destPath"
  ],
  "additionalProperties": false,
  "properties": {
    "format": {
      "enum": [
        "java_block_item_json",
        "gecko_geo_anim",
        "animated_java",
        "generic_model_json",
        "gltf",
        "native_codec"
      ]
    },
    "codecId": {
      "type": "string"
    },
    "destPath": {
      "type": "string"
    },
    "options": {
      "type": "object",
      "additionalProperties": false,
      "properties": {
        "fallback": {
          "type": "string",
          "enum": [
            "strict",
            "best_effort"
          ]
        },
        "includeDiagnostics": {
          "type": "boolean"
        }
      }
    },
    "includeState": {
      "type": "boolean"
    },
    "ifRevision": {
      "type": "string"
    }
  }
}
```

## 필드 레퍼런스
| Field | Type | 필수 | 설명 | 제약 |
| --- | --- | --- | --- | --- |
| `format` | `enum("java_block_item_json", "gecko_geo_anim", "animated_java", "generic_model_json", "gltf", "native_codec")` | ✅ | - | enum: "java_block_item_json", "gecko_geo_anim", "animated_java", "generic_model_json", "gltf", "native_codec" |
| `codecId` | `string` | ❌ | - | - |
| `destPath` | `string` | ✅ | - | - |
| `options` | `object` | ❌ | - | additionalProperties=false |
| `options.fallback` | `enum("strict", "best_effort")` | ❌ | - | enum: "strict", "best_effort" |
| `options.includeDiagnostics` | `boolean` | ❌ | - | - |
| `includeState` | `boolean` | ❌ | - | - |
| `ifRevision` | `string` | ❌ | - | - |

## 최소 요청 예시
```json
{
  "format": "java_block_item_json",
  "destPath": "./out/model.json"
}
```

## 응답 예시
```json
{
  "structuredContent": {
    "path": "value"
  }
}
```

## 운영 노트
- 변경 도구는 `ifRevision` 사용을 권장합니다.
- 스키마에 없는 필드는 보내지 마세요 (strict schema).
- `validate`로 최종 상태를 확인한 후 `export`를 수행하세요.
