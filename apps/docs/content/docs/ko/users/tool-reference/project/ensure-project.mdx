## 빠른 요약
활성 프로젝트를 재사용하거나 조건에 맞춰 생성/정리해 작업 가능한 상태를 보장합니다.

## 개요
- 도구 이름: `ensure_project`
- 카테고리: 프로젝트
- Strict schema: `true`

## 도구 설명
활성 프로젝트를 재사용하거나 조건에 맞춰 생성/정리해 작업 가능한 상태를 보장합니다.

## 입력 스키마 (JSON)
```json
{
  "type": "object",
  "additionalProperties": false,
  "properties": {
    "action": {
      "type": "string",
      "enum": [
        "ensure",
        "delete"
      ]
    },
    "target": {
      "type": "object",
      "additionalProperties": false,
      "properties": {
        "name": {
          "type": "string"
        }
      }
    },
    "format": {
      "type": "string",
      "enum": [
        "Java Block/Item",
        "geckolib",
        "animated_java",
        "Generic Model"
      ]
    },
    "name": {
      "type": "string"
    },
    "match": {
      "type": "string",
      "enum": [
        "none",
        "format",
        "name",
        "format_and_name"
      ]
    },
    "onMismatch": {
      "type": "string",
      "enum": [
        "reuse",
        "error",
        "create"
      ]
    },
    "onMissing": {
      "type": "string",

        "create",
        "error"
      ]
    },

      "type": "boolean"
```
