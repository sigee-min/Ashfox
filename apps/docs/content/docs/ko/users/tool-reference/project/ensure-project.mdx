## 빠른 요약
활성 프로젝트를 재사용하거나 조건에 맞춰 생성/정리해 작업 가능한 상태를 보장합니다.

## 개요
- 도구 이름: `ensure_project`
- 카테고리: 프로젝트
- Strict schema: `true`

## 도구 설명
활성 프로젝트를 재사용하거나 조건에 맞춰 생성/정리해 작업 가능한 상태를 보장합니다.

## 입력 스키마 (JSON)
```json

    "target": {
      "type": "object",

      }

        "Java Block/Item",

        "Generic Model"

    "match": {
```

## 운영 노트
- 변경 도구는 `ifRevision` 사용을 권장합니다.
- 스키마에 없는 필드는 보내지 마세요 (strict schema).
- `validate`로 최종 상태를 확인한 후 `export`를 수행하세요.
