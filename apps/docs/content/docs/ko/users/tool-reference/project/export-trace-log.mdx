## 빠른 요약
최근 실행 흐름의 진단 로그를 내보내 문제 분석에 활용합니다.

## 개요
- 도구 이름: `export_trace_log`
- 카테고리: 프로젝트
- Strict schema: `true`

## 도구 설명
최근 실행 흐름의 진단 로그를 내보내 문제 분석에 활용합니다.

## 입력 스키마 (JSON)
```json
{
  "type": "object",
  "additionalProperties": false,
  "properties": {
    "mode": {
      "type": "string",
      "enum": [
        "auto",
        "writeFile",
        "export"
      ]
    },
    "destPath": {
      "type": "string"
    },
    "fileName": {
      "type": "string"
    }
  }
}
```

## 필드 레퍼런스
| Field | Type | 필수 | 설명 | 제약 |
| --- | --- | --- | --- | --- |
| `mode` | `enum("auto", "writeFile", "export")` | ❌ | - | enum: "auto", "writeFile", "export" |
| `destPath` | `string` | ❌ | - | - |
| `fileName` | `string` | ❌ | - | - |

## 운영 노트

- `validate`로 최종 상태를 확인한 후 `export`를 수행하세요.
