## 빠른 요약
현재 프로젝트를 렌더링해 시각 결과를 빠르게 확인합니다.

## 개요
- 도구 이름: `render_preview`
- 카테고리: 프로젝트
- Strict schema: `true`

## 도구 설명
현재 프로젝트를 렌더링해 시각 결과를 빠르게 확인합니다.

## 입력 스키마 (JSON)
```json
{
  "type": "object",
  "required": [
    "mode"
  ],
  "additionalProperties": false,
  "properties": {
    "mode": {
      "enum": [
        "fixed",
        "turntable"
      ]
    },
    "angle": {
      "type": "array",
      "items": {
        "type": "number"
      },
      "minItems": 2,
      "maxItems": 3
    },
    "clip": {
      "type": "string"
    },
    "timeSeconds": {
      "type": "number"
    },
    "durationSeconds": {
      "type": "number"
    },
    "fps": {
      "type": "number"
    },
    "output": {
      "enum": [
        "single",
        "sequence"
      ]
    },
    "saveToTmp": {
      "type": "boolean"
    },
    "tmpName": {
      "type": "string"
    },
    "tmpPrefix": {
      "type": "string"
    },
    "includeState": {
      "type": "boolean"
    },
    "ifRevision": {
      "type": "string"
    }
  }
}
```

## 필드 레퍼런스
| Field | Type | 필수 | 설명 | 제약 |
| --- | --- | --- | --- | --- |
| `mode` | `enum("fixed", "turntable")` | ✅ | - | enum: "fixed", "turntable" |
| `angle` | `array<number>` | ❌ | - | minItems: 2, maxItems: 3 |
| `clip` | `string` | ❌ | - | - |
| `timeSeconds` | `number` | ❌ | - | - |
| `durationSeconds` | `number` | ❌ | - | - |
| `fps` | `number` | ❌ | - | - |
| `output` | `enum("single", "sequence")` | ❌ | - | enum: "single", "sequence" |
| `saveToTmp` | `boolean` | ❌ | - | - |
| `tmpName` | `string` | ❌ | - | - |
| `tmpPrefix` | `string` | ❌ | - | - |
| `includeState` | `boolean` | ❌ | - | - |
| `ifRevision` | `string` | ❌ | - | - |

## 최소 요청 예시
```json
{
  "mode": "fixed"
}
```

## 응답 예시
```json
{
  "structuredContent": {
    "kind": "single",
    "frameCount": 0
  }
}
```

## 운영 노트
- 변경 도구는 `ifRevision` 사용을 권장합니다.
- 스키마에 없는 필드는 보내지 마세요 (strict schema).
- `validate`로 최종 상태를 확인한 후 `export`를 수행하세요.
