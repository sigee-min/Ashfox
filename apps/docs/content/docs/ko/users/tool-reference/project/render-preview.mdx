## 빠른 요약
현재 프로젝트를 렌더링해 시각 결과를 빠르게 확인합니다.

## 개요
- 도구 이름: `render_preview`
- 카테고리: 프로젝트
- Strict schema: `true`

## 도구 설명
현재 프로젝트를 렌더링해 시각 결과를 빠르게 확인합니다.

## 입력 스키마 (JSON)
```json
{
  "type": "object",
  "required": [
    "mode"
  ],
  "additionalProperties": false,
  "properties": {
    "mode": {
      "enum": [
        "fixed",
        "turntable"
      ]
    },
    "angle": {
      "type": "array",
      "items": {
        "type": "number"
      },
      "minItems": 2,
      "maxItems": 3
    },
    "clip": {
      "type": "string"
    },
    "timeSeconds": {
      "type": "number"
    },
    "durationSeconds": {
      "type": "number"
    },
    "fps": {
      "type": "number"
    },
    "output": {
      "enum": [
        "single",
        "sequence"
      ]
    },
    "saveToTmp": {
      "type": "boolean"
    },
    "tmpName": {
      "type": "string"
    },
    "tmpPrefix": {
      "type": "string"
    },
    "includeState": {
      "type": "boolean"
    },
    "ifRevision": {
      "type": "string"
    }
  }
}
```

## 필드 레퍼런스
| Field | Type | 필수 | 설명 | 제약 |
| --- | --- | --- | --- | --- |
| `mode` | `enum("fixed", "turntable")` | ✅ | 스키마에 별도 설명이 없습니다. | enum: "fixed", "turntable" |
| `angle` | `array<number>` | ❌ | 스키마에 별도 설명이 없습니다. | minItems: 2, maxItems: 3 |
| `clip` | `string` | ❌ | 스키마에 별도 설명이 없습니다. | 추가 제약이 선언되지 않았습니다. |
| `timeSeconds` | `number` | ❌ | 스키마에 별도 설명이 없습니다. | 추가 제약이 선언되지 않았습니다. |
| `durationSeconds` | `number` | ❌ | 스키마에 별도 설명이 없습니다. | 추가 제약이 선언되지 않았습니다. |
| `fps` | `number` | ❌ | 스키마에 별도 설명이 없습니다. | 추가 제약이 선언되지 않았습니다. |
| `output` | `enum("single", "sequence")` | ❌ | 스키마에 별도 설명이 없습니다. | enum: "single", "sequence" |
| `saveToTmp` | `boolean` | ❌ | 스키마에 별도 설명이 없습니다. | 추가 제약이 선언되지 않았습니다. |
| `tmpName` | `string` | ❌ | 스키마에 별도 설명이 없습니다. | 추가 제약이 선언되지 않았습니다. |
| `tmpPrefix` | `string` | ❌ | 스키마에 별도 설명이 없습니다. | 추가 제약이 선언되지 않았습니다. |
| `includeState` | `boolean` | ❌ | 스키마에 별도 설명이 없습니다. | 추가 제약이 선언되지 않았습니다. |
| `ifRevision` | `string` | ❌ | 스키마에 별도 설명이 없습니다. | 추가 제약이 선언되지 않았습니다. |

## 최소 요청 예시
```json
{
  "mode": "fixed"
}
```

## 응답 예시
이 응답은 스키매틱 예시입니다. 실제 값과 선택 필드는 프로젝트 상태 및 런타임 버전에 따라 달라질 수 있습니다.

```json
{
  "structuredContent": {
    "kind": "single",
    "frameCount": 0
  }
}
```

## 운영 노트
- 입력 스키마에 선언된 필드만 전송하세요.
- 응답 예시는 스키매틱이므로 선택 필드는 방어적으로 처리하세요.
- 변경 도구에서 지원되는 경우 `ifRevision`을 사용해 충돌을 줄이세요.
