## 빠른 요약
선택한 큐브/페이스에 텍스처를 연결합니다.

## 개요
- 도구 이름: `assign_texture`
- 카테고리: 텍스처
- Strict schema: `true`

## 도구 설명
선택한 큐브/페이스에 텍스처를 연결합니다.

## 입력 스키마 (JSON)
```json
{
  "type": "object",
  "additionalProperties": false,
  "properties": {
    "textureId": {
      "type": "string"
    },
    "textureName": {
      "type": "string"
    },
    "cubeIds": {
      "type": "array",
      "items": {
        "type": "string"
      },
      "description": "Limit to these cube ids. If cubeNames is also provided, both must match."
    },
    "cubeNames": {
      "type": "array",
      "items": {
        "type": "string"
      },
      "description": "Limit to these cube names. If cubeIds is also provided, both must match."
    },
    "faces": {
      "type": "array",
      "minItems": 1,
      "items": {
        "type": "string",
        "enum": [
          "north",
          "south",
          "east",
          "west",
          "up",
          "down"
        ]
      }
    },
    "ifRevision": {
      "type": "string"
    },
    "includeState": {
      "type": "boolean"
    },
    "includeDiff": {
      "type": "boolean"
    },
    "diffDetail": {
      "type": "string",
      "enum": [
        "summary",
        "full"
      ]
    }
  }
}
```

## 필드 레퍼런스
| Field | Type | 필수 | 설명 | 제약 |
| --- | --- | --- | --- | --- |
| `textureId` | `string` | ❌ | 스키마에 별도 설명이 없습니다. | 추가 제약이 선언되지 않았습니다. |
| `textureName` | `string` | ❌ | 스키마에 별도 설명이 없습니다. | 추가 제약이 선언되지 않았습니다. |
| `cubeIds` | `array<string>` | ❌ | Limit to these cube ids. If cubeNames is also provided, both must match. | 추가 제약이 선언되지 않았습니다. |
| `cubeNames` | `array<string>` | ❌ | Limit to these cube names. If cubeIds is also provided, both must match. | 추가 제약이 선언되지 않았습니다. |
| `faces` | `array<enum("north", "south", "east", "west", "up", "down")>` | ❌ | 스키마에 별도 설명이 없습니다. | minItems: 1 |
| `ifRevision` | `string` | ❌ | 스키마에 별도 설명이 없습니다. | 추가 제약이 선언되지 않았습니다. |
| `includeState` | `boolean` | ❌ | 스키마에 별도 설명이 없습니다. | 추가 제약이 선언되지 않았습니다. |
| `includeDiff` | `boolean` | ❌ | 스키마에 별도 설명이 없습니다. | 추가 제약이 선언되지 않았습니다. |
| `diffDetail` | `enum("summary", "full")` | ❌ | 스키마에 별도 설명이 없습니다. | enum: "summary", "full" |

## 최소 요청 예시
```json
{
  "textureName": "texture",
  "cubeNames": [
    "cube"
  ]
}
```

## 응답 예시
이 응답은 스키매틱 예시입니다. 실제 값과 선택 필드는 프로젝트 상태 및 런타임 버전에 따라 달라질 수 있습니다.

```json
{
  "structuredContent": {
    "textureName": "value",
    "cubeCount": 0
  }
}
```

## 운영 노트
- 입력 스키마에 선언된 필드만 전송하세요.
- 응답 예시는 스키매틱이므로 선택 필드는 방어적으로 처리하세요.
- 변경 도구에서 지원되는 경우 `ifRevision`을 사용해 충돌을 줄이세요.
