## 빠른 요약
사운드/파티클/타임라인 트리거 키를 설정합니다.

## 개요
- 도구 이름: `set_trigger_keyframes`
- 카테고리: 애니메이션
- Strict schema: `true`

## 도구 설명
사운드/파티클/타임라인 트리거 키를 설정합니다.

## 입력 스키마 (JSON)
```json
{
  "type": "object",
  "required": [
    "clip",
    "channel",
    "keys"
  ],
  "additionalProperties": false,
  "properties": {
    "clipId": {
      "type": "string"
    },
    "clip": {
      "type": "string"
    },
    "channel": {
      "type": "string",
      "enum": [
        "sound",
        "particle",
        "timeline"
      ]
    },
    "keys": {
      "type": "array",
      "minItems": 1,
      "maxItems": 1,
      "items": {
        "type": "object",
        "required": [
          "time",
          "value"
        ],
        "additionalProperties": false,
        "properties": {
          "time": {
            "type": "number"
          },
          "value": {}
        }
      }
    },
    "ifRevision": {
      "type": "string"
    },
    "includeState": {
      "type": "boolean"
    },
    "includeDiff": {
      "type": "boolean"
    },
    "diffDetail": {
      "type": "string",
      "enum": [
        "summary",
        "full"
      ]
    }
  }
}
```

## 필드 레퍼런스
| Field | Type | 필수 | 설명 | 제약 |
| --- | --- | --- | --- | --- |
| `clipId` | `string` | ❌ | - | - |
| `clip` | `string` | ✅ | - | - |
| `channel` | `enum("sound", "particle", "timeline")` | ✅ | - | enum: "sound", "particle", "timeline" |
| `keys` | `array<object>` | ✅ | - | minItems: 1, maxItems: 1 |
| `keys[].time` | `number` | ✅ | - | - |
| `keys[].value` | `any` | ✅ | - | - |
| `ifRevision` | `string` | ❌ | - | - |
| `includeState` | `boolean` | ❌ | - | - |
| `includeDiff` | `boolean` | ❌ | - | - |
| `diffDetail` | `enum("summary", "full")` | ❌ | - | enum: "summary", "full" |

## 최소 요청 예시
```json
{
  "clip": "clip",
  "channel": "timeline",
  "keys": [
    {
      "time": 0,
      "value": "marker"
    }
  ]
}
```

## 응답 예시
```json
{
  "structuredContent": {
    "clip": "value",
    "channel": "value"
  }
}
```

## 운영 노트
- 변경 도구는 `ifRevision` 사용을 권장합니다.
- 스키마에 없는 필드는 보내지 마세요 (strict schema).
- `validate`로 최종 상태를 확인한 후 `export`를 수행하세요.
