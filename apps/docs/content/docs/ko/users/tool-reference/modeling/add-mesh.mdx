## 빠른 요약
새 메시를 추가합니다.

## 개요
- 도구 이름: `add_mesh`
- 카테고리: 모델링
- Strict schema: `true`

## 도구 설명
새 메시를 추가합니다.

## 입력 스키마 (JSON)
```json
{
  "type": "object",
  "required": [
    "name",
    "vertices",
    "faces"
  ],
  "additionalProperties": false,
  "properties": {
    "id": {
      "type": "string"
    },
    "name": {
      "type": "string"
    },
    "bone": {
      "type": "string"
    },
    "boneId": {
      "type": "string"

      "type": "array",
      "items": {
        "type": "number"
      },
      "minItems": 3,
      "maxItems": 3
    },
    "rotation": {
      "type": "array",
      "items": {
        "type": "number"
      },
      "minItems": 3,
      "maxItems": 3
    },

    },
    "uvPolicy": {
      "type": "object",
      "additionalProperties": false,
      "properties": {
        "symmetryAxis": {
          "type": "string",
          "enum": [
            "none",
            "x",
            "y",
            "z"
          ]
        },
        "texelDensity": {
          "type": "number",
          "minimum": 0.25,
          "maximum": 64
        },
        "padding": {
          "type": "number",
          "minimum": 0,

        }
      }

    "vertices": {
      "type": "array",
      "minItems": 3,
      "items": {
        "type": "object",
        "required": [
          "id",
          "pos"
        ],
        "additionalProperties": false,
        "properties": {
          "id": {
            "type": "string"
          },
          "pos": {
            "type": "array",

              "type": "number"

            "minItems": 3,

        }

    },
    "faces": {
      "type": "array",
      "minItems": 1,
      "items": {
        "type": "object",
        "required": [
          "vertices"
        ],

        "properties": {
          "id": {
            "type": "string"
          },
          "vertices": {

  }
```

## 필드 레퍼런스
| Field | Type | 필수 | 설명 | 제약 |
| --- | --- | --- | --- | --- |
| `id` | `string` | ❌ | 스키마에 별도 설명이 없습니다. | 추가 제약이 선언되지 않았습니다. |
| `name` | `string` | ✅ | 스키마에 별도 설명이 없습니다. | 추가 제약이 선언되지 않았습니다. |
| `bone` | `string` | ❌ | 스키마에 별도 설명이 없습니다. | 추가 제약이 선언되지 않았습니다. |
| `boneId` | `string` | ❌ | 스키마에 별도 설명이 없습니다. | 추가 제약이 선언되지 않았습니다. |
| `origin` | `array<number>` | ❌ | 스키마에 별도 설명이 없습니다. | minItems: 3, maxItems: 3 |
| `rotation` | `array<number>` | ❌ | 스키마에 별도 설명이 없습니다. | minItems: 3, maxItems: 3 |
| `visibility` | `boolean` | ❌ | 스키마에 별도 설명이 없습니다. | 추가 제약이 선언되지 않았습니다. |

| `uvPolicy.symmetryAxis` | `enum("none", "x", "y", "z")` | ❌ | 스키마에 별도 설명이 없습니다. | enum: "none", "x", "y", "z" |
| `uvPolicy.texelDensity` | `number` | ❌ | 스키마에 별도 설명이 없습니다. | min: 0.25, max: 64 |
| `uvPolicy.padding` | `number` | ❌ | 스키마에 별도 설명이 없습니다. | min: 0, max: 16 |

| `vertices[].pos` | `array<number>` | ✅ | 스키마에 별도 설명이 없습니다. | minItems: 3, maxItems: 3 |
| `faces` | `array<object>` | ✅ | 스키마에 별도 설명이 없습니다. | minItems: 1 |
| `faces[].id` | `string` | ❌ | 스키마에 별도 설명이 없습니다. | 추가 제약이 선언되지 않았습니다. |
| `faces[].vertices` | `array<string>` | ✅ | 스키마에 별도 설명이 없습니다. | minItems: 3 |
| `faces[].texture` | `string \ | enum(false)` | ❌ | 추가 제약이 선언되지 않았습니다. |
| `ifRevision` | `string` | ❌ | 스키마에 별도 설명이 없습니다. | 추가 제약이 선언되지 않았습니다. |
| `includeState` | `boolean` | ❌ | 스키마에 별도 설명이 없습니다. | 추가 제약이 선언되지 않았습니다. |
| `includeDiff` | `boolean` | ❌ | 스키마에 별도 설명이 없습니다. | 추가 제약이 선언되지 않았습니다. |
| `diffDetail` | `enum("summary", "full")` | ❌ | 스키마에 별도 설명이 없습니다. | enum: "summary", "full" |

## 최소 요청 예시
```json
{
  "name": "mesh",
  "vertices": [
    {
      "id": "v0",
      "pos": [
        0,
        0,
        0
      ]
    },
    {
      "id": "v1",
      "pos": [
        1,
        0,
        0
      ]
    },
    {
      "id": "v2",
      "pos": [
        0,
        1,
        0
      ]
    }
  ],
  "faces": [
    {
      "vertices": [
        "v0",
        "v1",
        "v2"
      ]
    }
  ]
}
```

## 응답 예시
이 응답은 스키매틱 예시입니다. 실제 값과 선택 필드는 프로젝트 상태 및 런타임 버전에 따라 달라질 수 있습니다.

```json
{
  "structuredContent": {
    "id": "value",
    "name": "value"
  }
}
```

## 운영 노트
- 입력 스키마에 선언된 필드만 전송하세요.
- 응답 예시는 스키매틱이므로 선택 필드는 방어적으로 처리하세요.
- 변경 도구에서 지원되는 경우 `ifRevision`을 사용해 충돌을 줄이세요.
