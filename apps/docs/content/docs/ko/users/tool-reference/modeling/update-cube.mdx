## 빠른 요약
기존 큐브의 형상/변환/소속 정보를 갱신합니다.

## 개요
- 도구 이름: `update_cube`
- 카테고리: 모델링
- Strict schema: `true`

## 도구 설명
기존 큐브의 형상/변환/소속 정보를 갱신합니다.
