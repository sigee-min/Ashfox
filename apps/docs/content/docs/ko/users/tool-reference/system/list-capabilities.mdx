## 빠른 요약
런타임에서 사용 가능한 포맷, 제한치, 기능 플래그를 조회합니다.

## 개요
- 도구 이름: `list_capabilities`
- 카테고리: 시스템
- Strict schema: `true`

## 도구 설명
런타임에서 사용 가능한 포맷, 제한치, 기능 플래그를 조회합니다.

## 입력 스키마 (JSON)
```json
{
  "type": "object",
  "additionalProperties": false
}
```

## 필드 레퍼런스
입력 필드가 없습니다.

## 최소 요청 예시
```json
{}
```

## 응답 예시
```json
{
  "structuredContent": {
    "pluginVersion": "value",
    "blockbenchVersion": "value",
    "formats": [
      {
        "format": "Java Block/Item",
        "animations": false,
        "enabled": false
      }
    ],
    "limits": {
      "maxCubes": 0,
      "maxTextureSize": 0,
      "maxAnimationSeconds": 0
    }
  }
}
```

## 운영 노트
- 변경 도구는 `ifRevision` 사용을 권장합니다.
- 스키마에 없는 필드는 보내지 마세요 (strict schema).
- `validate`로 최종 상태를 확인한 후 `export`를 수행하세요.
