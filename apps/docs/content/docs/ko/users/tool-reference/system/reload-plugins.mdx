## 빠른 요약
플러그인 런타임을 재로드해 변경된 확장 로직을 즉시 반영합니다.

## 개요
- 도구 이름: `reload_plugins`
- 카테고리: 시스템
- Strict schema: `true`

## 도구 설명
플러그인 런타임을 재로드해 변경된 확장 로직을 즉시 반영합니다.

## 입력 스키마 (JSON)
```json
{
  "type": "object",
  "required": [
    "confirm"
  ],
  "additionalProperties": false,
  "properties": {
    "confirm": {
      "type": "boolean"
    },
    "delayMs": {
      "type": "number"
    }
  }
}
```

## 필드 레퍼런스
| Field | Type | 필수 | 설명 | 제약 |
| --- | --- | --- | --- | --- |
| `confirm` | `boolean` | ✅ | - | - |
| `delayMs` | `number` | ❌ | - | - |

## 최소 요청 예시
```json
{
  "confirm": true
}
```

## 응답 예시
```json
{
  "structuredContent": {
    "scheduled": true,
    "delayMs": 0,
    "method": "devReload"
  }
}
```

## 운영 노트
- 변경 도구는 `ifRevision` 사용을 권장합니다.
- 스키마에 없는 필드는 보내지 마세요 (strict schema).
- `validate`로 최종 상태를 확인한 후 `export`를 수행하세요.
