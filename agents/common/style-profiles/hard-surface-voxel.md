# Style Profile: Hard Surface Voxel

## Intent
Crisp engineered look with strong panel logic and controlled edge highlights.

## Palette Rules
- Narrow palette with disciplined accent usage.
- Neutral base tones plus one accent family.

## Shading Rules
- Favor clean edge highlights and panel separation.
- Keep gradients minimal; use discrete tonal regions.
- Use contrast to separate functional components.

## Geometry Rules
- Align features to clear mechanical intent.
- Avoid ambiguous bevels that read as accidental.

## Failure Signals
- Muddy palette with too many unrelated hues.
- Unclear panel boundaries.
- Highlight placement that conflicts with chosen light direction.
